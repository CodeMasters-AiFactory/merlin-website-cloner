# Legal Guidelines

## Overview

Merlin Website Clone is designed to help users create offline backups of websites they have legal rights to backup. This document outlines legal guidelines and best practices.

## User Responsibilities

### Legal Rights
- Only clone websites you own or have explicit permission to clone
- Respect copyright and intellectual property rights
- Comply with website Terms of Service
- Follow robots.txt directives
- Respect rate limits and server resources

### Prohibited Uses
- Cloning websites without permission
- Violating Terms of Service
- Bypassing security measures for unauthorized access
- Using cloned content for commercial purposes without permission
- Violating copyright or intellectual property rights

## robots.txt Compliance

Merlin Website Clone automatically checks and respects robots.txt files:

- **Automatic Checking**: Before scraping, the tool checks for robots.txt
- **Rule Enforcement**: Disallowed paths are automatically skipped
- **Crawl Delay**: Respects specified crawl delays
- **User-Agent**: Uses appropriate user-agent strings

## Rate Limiting

The tool includes built-in rate limiting to prevent server overload:

- **Configurable Limits**: Set requests per second/minute/hour
- **Automatic Throttling**: Automatically slows down when limits are reached
- **Respectful Scraping**: Default settings are conservative

## Terms of Service

Always review and comply with a website's Terms of Service before cloning:

- **Read ToS**: Check the website's Terms of Service
- **Respect Restrictions**: Follow any restrictions on automated access
- **Commercial Use**: Verify if commercial use of cloned content is allowed

## Best Practices

1. **Get Permission**: Always get explicit permission when possible
2. **Respect robots.txt**: Always respect robots.txt directives
3. **Rate Limiting**: Use conservative rate limits
4. **Personal Use**: Cloned content should be for personal backup only
5. **No Redistribution**: Don't redistribute cloned content without permission

## Disclaimer

Merlin Website Clone is a tool. Users are responsible for ensuring their use complies with all applicable laws and regulations. The developers are not responsible for misuse of this tool.

## Support

For legal questions, consult with a legal professional familiar with web scraping and copyright law in your jurisdiction.
